# 📊 Customer Churn Prediction & Business Insights

An end-to-end Data Analytics and Machine Learning project that analyzes customer behavior and predicts customer churn using Python and Machine Learning algorithms.

The objective of this project is to identify customers who are likely to leave a telecom company and uncover the key factors influencing churn. The insights generated can help businesses improve customer retention strategies and reduce revenue loss.

---

## 🚀 Project Overview

Customer churn is one of the biggest challenges faced by subscription-based businesses. Retaining an existing customer is significantly more cost-effective than acquiring a new one.

In this project, I performed:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Business Insight Generation
- Feature Engineering
- Machine Learning Model Building
- Model Evaluation
- Feature Importance Analysis
- Model Saving for Future Deployment

---

## 📂 Dataset

The dataset contains customer information such as:

- Customer Demographics
- Tenure
- Monthly Charges
- Total Charges
- Contract Type
- Internet Service
- Payment Method
- Additional Services
- Churn Status (Target Variable)

## 🎯 Project Objectives

- Understand customer behavior through Exploratory Data Analysis.
- Identify the major factors contributing to customer churn.

# 📈 Exploratory Data Analysis

The EDA focuses on answering important business questions such as:

- What percentage of customers churn?
- Which contract types have the highest churn?
- How does tenure affect churn?
- Does monthly billing influence churn?
- Which additional services reduce churn?
- Which customer segments are most likely to leave?

Several visualizations were created to identify meaningful business patterns and customer behavior.

---

# 🤖 Machine Learning Models

The following classification algorithms were implemented and compared:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier
- XGBoost Classifier

Each model was evaluated using multiple performance metrics.

---

# 📊 Evaluation Metrics

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
- Confusion Matrix *(if included)*

---

# 💡 Business Insights

Some key observations from the analysis include:

- Customers with month-to-month contracts have a higher churn rate.
- Customers with shorter tenure are more likely to leave.
- Higher monthly charges are associated with increased churn.
- Customers using additional services such as Online Security and Tech Support are less likely to churn.
- Long-term contracts significantly improve customer retention.

These insights can help businesses design targeted retention campaigns and improve customer satisfaction.
